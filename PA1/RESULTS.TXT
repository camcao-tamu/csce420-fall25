H2 - Advanced Heuristic

My heuristic works based off of a penalty system given based on the block's current position comparitively to the goal position.

2 points if it is in the WRONG STACK
1 point if it is in the WRONG POSITION but CORRECT STACK
1 point if it is in the CORRECT POSITION AND CORRECT STACK but has an incorrect block above it (1 point per block)


Example 1 (probA03 start):

Current State:
>>>>>>>>>>
CE
AD
B
>>>>>>>>>>
Goal State:
>>>>>>>>>>

ADBC
E
>>>>>>>>>>

Block C - Wrong stack, 2
Block E - Wrong stack, 2
Block A - Correct stack, Correct position, No incorrectly placed blocks above, 0
Block D - Correct stack, Correct position, 0
Block B - Wrong stack, 2
Total = 2 + 2 + 0 + 0 + 2 = 6
